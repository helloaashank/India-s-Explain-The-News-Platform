# Project Brief

Bharat Brief is a premium Indian news explainer platform.

The product goal is to make important news understandable in two minutes. Instead of copying the structure of a newspaper, every story is explained in plain language using a repeatable framework.

## Core Article Framework

Every article answers:

1. What happened?
2. Why does it matter?
3. Who is affected?
4. What happens next?

This framework is already represented in the current `Article` type inside `data/news.ts`.

## Target Users

Primary users:

- Indian mobile readers who want quick context.
- Students preparing for exams like UPSC.
- Users who follow government schemes, business, policy, sports, technology, and international affairs.
- Readers who prefer clarity over traditional newspaper density.

## Product Feeling

The intended product feeling is:

- Clean like Apple.
- Simple like Notion.
- Intelligent like Perplexity.
- Credible like Ground News.
- Fast to scan like Inshorts.

The user should feel: "I finally understand the news."

## Design System

Current color direction:

- Background: `#FFFFFF`
- Primary: `#111111`
- Secondary: `#666666`
- Accent: `#2563EB`
- Success: `#16A34A`
- Warning: `#EA580C`
- Border: `#ECECEC`

Current UI principles:

- Large whitespace.
- Rounded corners.
- Soft shadows.
- Minimal color usage.
- Mobile-first layout.
- Premium, clearly marked ads.
